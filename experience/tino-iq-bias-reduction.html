TINO IQ · Data Science Intern · May 2020 — Aug 2020

Cyclic Boosting for interpretable tabular models

Python · NumPy · Wick et al., arXiv:2002.03425

I reproduced the Cyclic Boosting algorithm from Felix Wick, Ulrich Kerzel, and Michael Feindt, Cyclic Boosting — an explainable supervised machine learning algorithm (arXiv:2002.03425; originally ICMLA 2019). The method builds an additive supervised model by cycling over dimensions so individual predictions stay traceable—addressing the paper’s goal of accuracy without the black-box behavior of typical deep or ensemble stacks.

At TINO IQ the business question was tabular prediction where stakeholders still needed to see how inputs drive outputs, including situations where opaque scores complicate fairness review. An ordinary linear regression pipeline was the natural baseline: quick to ship but shallow for interaction structure and harder to extend when accuracy stalled.

I translated the paper’s cyclic fitting recipe into a NumPy-first Python trainer: successive rounds update dimension-wise (weak) contributions while preserving the separable additive form described in the paper. That kept the implementation aligned with the published procedure rather than a hand-wavy “boosted linear” sketch.

On the company’s prepared tabular datasets—same splits and evaluation protocol as the linear baseline—the Cyclic Boosting model improved accuracy by more than 10%. Training code used vectorized routines and batched updates so we could afford hyperparameter sweeps and ablations on internship timelines.
